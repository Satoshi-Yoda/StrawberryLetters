Для запуска требуется 64 bit OS Windows 7, 8, 10, 11?.

[левая кнопка мыши]
Щелчок на синей (не выделенной) снежинке выделяет её, при этом убирая предыдущее выделение.
Щелчок на выделенной (красной) снежинке расширяет выделение на 1 смежную снежинку во все стороны от выделенных по рёбрам.
Таким образом, чтобы выделить всю связанную группу снежинок, надо несколько раз пощёлкать по одной из снежинок.
При зажатом [shift] при щелчке на синюю снежинку добавляет её к текущему выделению.

[правая кнопка мыши]
Щелчок на пустом месте вызывает контекстное меню со всеми сохранёнными композициями и элементарными фигурами:
	- point (добавить точку)
	- honeycomb 2 (добавить honeycomb 2)
	- honeycomb 3 (добавить honeycomb 3)
	- transformer 3 (добавить transformer 3)
	- [имя].group (добавить сохранённую композицию с именем [имя])
	- remove saved group (удалить сохранённую композицию, требуется ввести её имя, без .group)

Щелчок на выделенной (красной) снежинке вызывает контекстное меню операций с выделенной группой снежинок:
	- save selection (вводим имя, нажимаем Enter, сохраняет выделенную группу на диск для последующего использования)
	- rotate CW (повернуть выделенное по часовой стрелке на 30 градусов)
	- rotate CCW (повернуть выделенное против часовой стрелки на 30 градусов)
	- flip (отразить выделенное по горизонтали)
	- around \ (обводит выделенную группу снаружи одним рядом снежинок, начиная с самой верхней точки, по часовой стрелке, отступив от неё в направлении вверх-влево)
	- around / (аналогично предыдущему, но с направлением вверх-вправо)

[стрелочки на клавиатуре]
При наличии выделенной группы (красных) снежинок перемещают выделенную группу на одно минимальное расстояние.
При отсутствии выделения (красных снежинок) перемещает камеру. При зажатии перемещает камеру непрерывно.

[delete] или [backspace]
Удаляет выделенные (красные) снежинки.

[-] и [+]
Уменьшает и увеличивает общий масштаб.
